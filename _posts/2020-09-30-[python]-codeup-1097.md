# 문제

부모님을 기다리던 영일이는 검정/흰 색 바둑알을 바둑판에 꽉 채워 깔아 놓고 놀다가...

"십(+)자 뒤집기를 해볼까?"하고 생각했다. ~~별 생각을 다 한다~~

바둑판(19 * 19)에 흰 돌(1) 또는 검정 돌(0)이 모두 꽉 채워져 놓여있을 때,
n개의 좌표를 입력받아 십(+)자 뒤집기한 결과를 출력하는 프로그램을 작성해보자.

* 참고 : 가로 번호, 세로 번호를 사용할 수 있는 2차원 배열을 사용하면 이러한 형태를 쉽게 기록하고 사용할 수 있다. 물론 더 확장한 n차원 배열도 만들 수 있다.

# 풀이
- 19x19에 0으로 가득한 2차원 리스트를 만들었다.

~~~Python
zero = [[0 for i in range(19)] for j in range(19)]
~~~

- 19번 돌면서 바둑판 상황을 한 줄 씩 받아보았다.

~~~Python
for i in range(19):
    zero[i] = list(map(int, input().split()))
~~~

- 좌표를 몇 개 찍을지 받아보았다.

~~~Python
num = int(input())
~~~

- 받아내고 싶은 좌표 갯수 만큼 돌면서 x,y 좌표를 받아보았다. 받아낸 좌표를 중심으로 바둑판을 19번 돌면서 0은 1로 1은 0으로 바꿔줬다.
  - x가 가로인지 y가 가로인지 슬라이싱 앞쪽에 넣어야 가로인지 슬라이싱 뒤쪽에 넣어야 가로인지 맨날 헷갈린다...
  - 내가 (3,3)을 찍고싶어도 컴퓨터는 0부터 숫자를 세기때문에 (4,4)를 찍게된다. 받아낸 좌표값에서 -1씩 해주자.

~~~Python
for i in range(num):
    x, y = map(int, input().split())
    for j in range(19):
        if zero[x-1][j] == 0:
            zero[x-1][j] = 1
        else:
            zero[x-1][j] = 0
        if zero[j][y-1] == 0:
            zero[j][y-1] = 1
        else:
            zero[j][y-1] = 0
~~~

- +자 뒤집기 된 바둑판을 print 해보자

~~~Python          
for i in range(19):
    for j in range(19):
        print(zero[i][j],end=' ')
    print('')
~~~
