
2. 시간적 응집도
3. 절차적 응집도
4. 교환적 응집도

### 3. 다음 중 객체지향 프로그래밍의 오버로딩(Overloading)과 오버라이딩(Overriding)에 관한 설명으로 옳지 __않은__ 것을 고르시오.

1. 오버로딩 : 메소드의 이름, 리턴 타입, 매개 변수 등이 동일해야한다.
2. 오버라이딩 : 상속에 의한 재정의를 말하며, 메소드의 이름, 리턴 타입, 매개 변수 등이 동일해야한다.
3. 오버로딩
~~~java
Class Overload{
  double add(int a, int b){
    return a + b;
  }
  double add(int a, double b){
    return a + b;
  }
}
~~~
4. 오버라이딩
~~~java
Class Overriding{
  int a;
  void addMain(){
    a = 10;
  }
}
Class Child extends Overriding{
  int data;
  void addMain(){
    data = 30;
  }
}
~~~

답 : 1
