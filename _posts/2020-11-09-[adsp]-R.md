# About R

- 무료 오픈소스 프로그램입니다.
  - 사용자 커뮤니티에 도움 요청이 쉽습니다.
  - 약 15,000개의 패키지가 수시로 업데이트됩니다.
  - 최신 알고리즘이 빠르게 반영됩니다.
  - 무료 문서, 학습 자료가 많습니다.
- 모든 운영체제에서 사용이 가능합니다.
- 통계 전문가들의 표준 플랫폼이 되고 있습니다. ~~vs Python~~
- ggplot2와 같은 쓸만한 그래픽 라이브러리가 있습니다. ~~vs matplotlib~~
- 시스템 기반 데이터 저장 방식입니다.
- 객체지향 언어이면서 함수형 언어입니다.
- GUI보다 CLI를 활용합니다.

## 특징

### 그래픽처리
- 고해상도의 그래프와 그림, 매우 세부적인 부분까지 사용자 지정 가능합니다.

### 데이터 처리 및 계산 능력
- 벡터, 행렬, 배열, 데이터 프레임, 리스트 등의 다양한 형태의 데이터 구조 지원합니다.
- 복잡한 구조 내의 개별 데이터에 접근하는 절차가 간단합니다.
- 큰 데이터 핸들링이 간편합니다.

### 패키지
- 스스로 개발하는 새로운 함수, 특정 분석 기법을 사용 할 수 있습니다.
- 확장성 -> 최근 이론이나 기법을 사용하기 용이합니다.
- 그래픽 처리가 용이합니다.
- install.packages("원하는패키지") # 패키지 설치
- library(원하는패키지) # 패키지 불러오기
### 객체지향 언어로서의 특징

- SPSS/SAS는 분석결과가 복잡하고 이해하고 사용하기 어렵습니다. 하지만 R은 결과를 담은 '객체'를 반환하므로 객체에서 필요한 부분만 뽑아 쓰면 되서 편리합니다.

### 함수형 언어로서의 특징

- 기존에 사용한 함수들을 활용하여 프로그래밍해서 더욱 깔끔하고 단축한 코드를 짤 수 있습니다.
- 코드 수행속도가 빠릅니다.
- 코드를 단순화하고 디버깅하기 쉽습니다.
- 병렬 프로그래밍(맵리듀스)로 전환하기가 다른 프로그램보다 쉽습니다.

## R 기초중에 기초

### 출력하기

- 커맨드 프롬프트에 변수나 표현식을 입력합니다.
- print() : 출력형식을 지정할 필요가 없습니다. 한번에 하나의 객체만 출력합니다.
- cat() : 여러 항목을 묶어서 연결된 결과로 출력, 복합적 데이터 구조(행렬, list 등)을 출력합니다.

### 변수에 값 할당하기

- <- (기본적으로 제일 많이 쓰임)
  - ex) a <- 3
- <<-, =, ->

### 변수 목록보기

- ls(), ls.str()

### 변수 삭제하기

- rm()
  - rm(list = ls()) : 모든 변수를 삭제 할 때 사용합니다.

## R 데이터 구조
> 단일값(Scalar),
> 요인(Factor) : 반복적인 값(학년, 성별 등),
> 벡터(Vector), 행렬(Matrix), 배열(Array),
> 리스트(List), 데이터프레임(Data Frame)

![R 데이터 종류](http://venus.ifca.unican.es/Rintro/_images/dataStructuresNew.png)


### 벡터(Vector)
- 하나의 스칼라 값 또는 하나 이상의 스칼라 원소를 갖는 단순한 형태의 집합입니다.
- 숫자, 문자, 논리연산자 등의 데이터를 넣을 수 있으며 동일한 자료형을 갖는 값들의 집합입니다.

- 벡터 생성하기 : c()
  - a <- c(1,2,3,4) # Numeric type
  - b <- c('a',2,3,4) # 이런 경우 Character형 벡터가 된다.

#### 기본적인 통계량 계산
  - 평균 : mean()
  - 중앙값 : median()
  - 표준편차[^1] : sd()
  - 분산 : var()
  - 상관계수 : cor()
  - 공분산 : cov()

#### 통계 기본
- 관측값에서 '평균' 또는 '중앙값을 빼서' '편차'를 구하고
- '편차'를 모두 더하면 0이 되니까 제곱해서 더한 후 전체 수로 나눠서 '분산'을 구하고
- '분산'을 제곱근해서 '표준 편차'를 구할 수 있다.



### 행렬(Matrix)
- data = 데이터 벡터, nrow = 행의 수, ncol = 열의 수, byrow = False(열 우선으로), dimnames = 각 차원의 이름

~~~R
m <- matrix(c(1,2,3,4,5,6), ncol = 2, byrow = FALSE, dimnames = NULL)
m
~~~

```
     [,1] [,2]
[1,]    1    4
[2,]    2    5
[3,]    3    6
```

### 배열(Array)

- 다차원 데이터 형태입니다.

### 데이터 프레임(Data Frame)

- 행렬과 유사한 2차원 구조입니다.
- 각 열이 서로 다른 데이터 타입을 가질 수 있습니다.
- 단, 벡터들의 길이가 반드시 같아야합니다.

### 리스트(List)

- 서로 다른 데이터 유형을 담을 수 있습니다.
