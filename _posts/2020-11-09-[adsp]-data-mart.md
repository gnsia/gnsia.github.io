# 데이터마트

![데이터마트](https://mblogthumb-phinf.pstatic.net/20131114_254/musasin84_1384404148546i42Bu_JPEG/%B5%A5%C0%CC%C5%CD%B8%B6%C6%AE.jpg?type=w2)

## 정의

- 데이터 웨하우스와 사용자 사이의 중간층에 위치합니다.
- 하나의 주제, 하나의 부서 중심으로 이루어진 데이터 웨어하우스라고 할 수 있습니다.
- 데이터 마트 내 대부분의 데이터는 데이터 웨어하우스로부터 복제됩니다.
- 자체적으로 데이터 수집도 가능합니다.
- 관계형 데이터 베이스나 다차원 데이터베이스를 이용하여 구축합니다.
