# 통계학 개론

## 통계 분석 개요

### 통계학의 정의

- 통계학은 자료로부터 유용한 정보를 이끌어내는 학문입니다.
- 자료의 수집, 정리, 해석 등의 방법이 있습니다.

---

### 모집단과 표본

- 모집단 : 관심의 대상이 되는 모든 개체의 집합
- 표본 : 모집단에서 분석 대상으로 관찰된 일부의 집합

---

### 표본 추출 방법

#### 확률적 추출

- 단순 무작위 추출 : 모집단 내 개별 개체가 선택될 확률이 모두 1/N로 일정
- 계통 추출 : 첫 번째 표본을 무작위로 선정 후 k번째 요소를 표본으로 선정하는 방식, 실질적으로 단순 무작위 추출과 거의 동일하다고 합니다.
- 층화 추출 : 모집단을 성격에 따라 몇 개의 집단 또는 층으로 나누고, 각 집단 내에서 원하는 크기의 표본을 무작위로 추출하는 방식입니다.
- 군집 추출 : 모집단을 여러개의 집단으로 나눈 뒤 선택한 집단 내에서 필요한 만큼의 표본을 임의로 선택하는 방식입니다.

#### 비확률적 추출

- 판단 추출 : 전문지식이 있는 연구자가 자신의 판단에 따라 표본을 선택하는 방식입니다.
- 할당 추출 : 모집단을 여러 집단으로 나눈 후 각 집단에서 필요한 개수의 표본을 선택하되 연구자가 자신의 판단에 따라 선택하는 경우입니다.
- 편의 추출 : 연구자가 쉽게 접근 할 수 있는 표본을 선택하는 경우입니다.

---

### 자료의 종류

- 명목 척도 : 단순히 측정 대상의 특성을 분류하거나 확인하기 위한 목적으로 숫자를 부여합니다.
  - ex) 성별
- 서열 척도 : 단순히 대소 또는 높고 낮음 등의 순위만 제공하며 양적 비교는 불가능 합니다.
  - ex) 상품에 대한 선호 순위, 1,2,3 순위 등
- 등간 척도 : 순위를 부여하되 순위 사이의 간격ㄱ이 동일하여 양적인 비교가 가능합니다. 단 절대 0점이 존재하지 않습니다.
  - ex) 온도계 수치, 물가지수, Likert 척도 등등
- 비율 척도 : 절대 0점이 존재하여 측정값 사이의 비율 계산이 가능한 척도입니다.
  - ex) 몸무게

## 통계 분석

### 기술 통계학

- 모집단 전체 혹은 표본으로부터 얻는 데이터에 대한 숫자 요약이나 그래프 요약을 통하여 데이터가 가진 정보를 정리하는 이론과 방법을 다룹니다.

### 추론 통계학

- 표본으로부터 얻은 정보를 이용하여 모집단의 특성(모수)을 추론하거나 변수들 가읜 함수 관계의 진위 여부를 판단하는 일련의 과정에 관한 이론과 방법을 다룹니다.

## 확률 및 확률 분포

### 확률의 정의

표본공간 S의 부분집합인 각 사상에 대하여 실수값을 가지는 함수 P가 다음의 세 가지 성질을 만족하면 확률이라 한답니다.

1. 모든 사건의 확률값은 0과 1사이에 있다.
2. 전체 집합의 확률은 1이다.
3. 서로 배반인 사건들 E1, E2 ... 의 합집합의 확률은 각 사건들의 확률의 합이다. 즉 위에서 배반사건이란 교집합이 공집합인 사건들을 말한다.

### 확률 변수와 확률 분포

- 확률 변수 : 표본공간에서 발생하는 원소를 정의역으로 하고 이에 대응되는 실수 값을 치역으로 하는 함수입니다.
  - 이산 확률 변수 : 변수가 취할 수 있는 값을 헤아려 열거 할 수 있을 때
  - 연속 확률 변수 : 주어진 실수 구간 내에 속하는 어떠한 실수도 취할 수 있을 때

- 확률 분포 : 치역에 해당되는 실수 값을 확률로 나타낸 것을 확률 분포하고 합니다.
  - 결합 확률 분포 : 두 확률 변수 X, Y의 모든 값과 이에 대응하는 확률을 표나 그림으로 나타낸 것을 X, Y의 결합 확률 분포라 합니다.
    - 이산형 확률 분포 : 베르누이 확률 분포, 이항 분포, 기하 분포, 다항 분포, 포아송 분포
    - 연속형 확률 분포 : 균일 부포, 정규 분포(카이제곱분포, F분포), 지수 분포

---

### 확률변수의 기댓값과 분산

- 확률 변수 X가 취하는 변화에 따라 확률 값들은 분포를 이루게 되며 이러한 확률 분포의 중심이 되는 평균을 기대값이라 부릅니다.
