# 연관규칙

## 연관규칙의 개념
- 상품의 구매, 서비스 등 일련의 거래, 사건들 간의 규칙을 발견하기 위해 적용합니다.
- 장바구니 분석 : 장바구니에 무엇이 같이 들어 있는지에 대한 분석이죠.
- 서열 분석 : A를 산 다음에 B를 산다.
- 조건과 반응의 형태 : 아메리카노를 마시는 손님 중 10%가 브라우니를 먹는다.

## 연관분석의 측도 3가지
- 지지도 : 전체 거래 중 항목 A, B를 동시에 포함하는 거래의 비율
- 신뢰도 : 항목 A를 포함한 거래중에서 항목 A, B가 같이 포함될 확률
- 향상도 : A가 주어지지 않았을 때 품목 B의 확률에 비해 A가 주어졌을 때의 품목 B의 확률의 증가 비율
  - 계산법 : A, B 동시에 포함된 거래수 / A거래수 * B거래수

## 연관규칙의 절차
- 최소지지도 결정 -> 최소지지도 넘는 품목 분류 -> 2가지 품목 집합 생성 -> 반복적으로 수행하여 빈발품목 집합 찾기

## 장점
- 탐색적 기법 : 쉽게 이해됩니다.
- 강력한 비목적성 분석 기법입니다.
- 사용 편리한 분석 데이터 형태를 띕니다.
- 계산이 용이합니다.

## 단점
- 상당한 수의 계산 과정 : 기하급수적으로 증가합니다.
  - 유사한 품목 범주화, 적은 연관규칙 제외로 해결
- 적절한 품목 결정 : 너무 세분화된 품목은 의미 없는 결과를 도출하기도 합니다.
  - 큰 범주 분석 후 세부 범주 분석으로 해결해야합니다.
- 품목의 비율차이 : 거래량 적으면 규칙 발견이 어려워집니다.
  - 유사한 품목과 함께 범주를 구성하여 해결합니다.

## 순차패턴
- 시간(구매시점)이라는 개념 포함시켜 순차적으로 구매 가능성이 큰 상품군을 찾아냅니다.

## 연관성분석 활용방안
- 장바구니 분석의 경우 실시간 상품 추천을 통한 교차판매에 응용합니다.
- 시차분석은 상품 추천하는 교차판매 캠페인에 사용합니다.
- 상품의 연관성을 분석하여 매장진열, 패키지 상품개발, 교차판매 전략의 수립에 활용합니다.
